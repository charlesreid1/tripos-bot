If all the values of $x$~and~$y$ given by
\[
ax^{2} + 2hxy + by^{2} = 1,\quad
a'x^{2} + 2h'xy + b'y^{2} = 1
\]
(where $a$,~$h$,~$b$, $a'$,~$h'$,~$b'$ are rational) are rational, then
\[
(h - h')^{2} - (a - a')(b - b'),\quad
(ab' - a'b)^{2} + 4(ah' - a'h)(bh' - b'h)
\]
are both squares of rational numbers. \MathTrip{1899.}
