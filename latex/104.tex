The first $n + 2$ terms in the expansion of $\log\left(1 + x + \dfrac{x^{2}}{2!} + \dots + \dfrac{x^{n}}{n!}\right)$ in
powers of~$x$ are
\[
x - \frac{x^{n+1}}{n!}
  \left\{\frac{1}{n + 1}
       - \frac{x}{1!\, (n + 2)}
       + \frac{x^{2}}{2!\, (n + 3)} - \dots
       + (-1)^{n} \frac{x^{n}}{n!\, (2n + 1)}
  \right\}.
\]
